Detailed Description

Function prototypes for the client notifications module.

This file contains the prototypes of the functions for the client notifications module. This module polls for notification messages sent by the server notifications module when an event occurs (I.E node registered in the network). Internal module

int tc_client_notifications_get ( unsigned int  timeout,
NET_MSG *  ret_msg 
)

Polls for a notification message.

Polls for a notification message sent by the server notifications module. This call is used by the client tc_client_get_node_event( unsigned int timeout, unsigned char *ret_event, unsigned int *ret_node_id ) call

Parameters
[in]timeoutMaximum time interval (in ms) to wait for a message. If 0 blocks indefinitely. Must be equal or greater than 0
[out]ret_msgThe buffer to store the polled message. Must not be a NULL pointer
Precondition
None
Returns
Upon successful return : ERR_OK (0)
Upon output error : An error code (<0)
Note
In the current version only node registration/unregistration events trigger a notification

Implement more notification events (I.E topic created/destroyed/changed)
