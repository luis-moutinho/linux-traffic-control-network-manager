Detailed Description

Function prototypes and implementation for the node database submodule of the servers database.

This submodule contains the functions to manage and handle the node related database of the servers database. Internal submodule

NODE_ENTRY* tc_server_db_node_create ( unsigned int  node_id)

Creates a new node entry.

Creates a new entry for the node. If an entry already exists it returns that entry address instead

Parameters
[in]node_idThe ID of the node for which the entry will be created. Must be equal or greater than 0
Precondition
assert( node_id );
Returns
Upon successful return : Address of the new/existing entry
Upon output error : A NULL pointer
