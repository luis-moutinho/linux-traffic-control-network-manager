Function prototypes for the sockets layer.

This file contains the function prototypes for the sockets layer used for control and topic comunications

int sock_receive ( SOCK_ENTITY *  sock,
SOCK_ENTITY *  unblock_sock,
unsigned int  timeout,
char *  ret_data,
unsigned int  buff_size,
NET_ADDR *  ret_sender 
)

Receives data from a socket.

Receives data through a connected/unconnected socket

Parameters
[in]sockThe socket to receive the data from. Must not be a NULL pointer
[in]unblock_sockThe socket from where to receive an unblock signal. Optional (can be a NULL pointer)
[in]timeoutThe maximum time interval (in ms) to wait for data. If 0 blocks indefinitely. Must be equal or greater than 0
[out]ret_dataThe buffer to store the received data. Must not be a NULL pointer
[in]buff_sizeThe size of the data buffer. Must be greater than 0 else a default size is considered (DEFAULT_MAX_SIZE)
[out]ret_senderThe buffer to store the address of the sender. Optional (can be a NULL pointer)
Precondition
None
Returns
Upon successful return : The number of received bytes
Upon output error : An error code (<0)
Note
The socket had to be previously bound and connected to the group address for multicast comunications. For unconnected sockets and point to point comunications dest has to be filled with the destination address.
