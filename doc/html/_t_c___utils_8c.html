Detailed Description

Source code of the utilities functions.

This file contains the implementation of the utilities functions. These functions aid in threads creation, NIC IP address retrieval and the transformation of data messages to host-to-network or network-to-host format

Creates a thread.

Creates and checks if the thread was created and is running. Creates feedback mutex if requested

Parameters
[in]thread_callThe thread code call. Must not be a NULL pointer
[out]ret_thread_idThe buffer to store the created thread ID. Must not be a NULL pointer
[out]ret_quit_flagThe address of the thread quit flag. Must not be a NULL pointer
[out]ret_thread_lockThe address of the thread mutex to be initialized (provides run status feedback). Optional (can be a NULL pointer)
[in]timeoutMaximum time interval (in ms) to wait for the thread to be running. Only used when a thread mutex is provided for feedback. Must be equal or greater than 0
Precondition
assert( thread_call );
assert( ret_thread_id );
assert( ret_quit_flag );
Returns
Upon successful return : ERR_OK (0)
Upon output error : An error code (<0)
Note
The thread should follow the follow structure
void thread_call( void )
{
pthread_mutex_lock( &thread_mutex );
while ( quit_flag == THREAD_RUN ){
//do stuff
}
pthread_mutex_unlock( &thread_mutex );
pthread_exit(NULL);
}

Destroys a thread.

Flags thread to stop and waits for it to end. Destroys feedback mutex if provided

Parameters
[in]thread_idThe ID of the thread to stop. Must not be a NULL pointer
[in]quit_flagThe address of the thread quit flag. Must not be a NULL pointer
[in]thread_lockThe address of the thread mutex to be destroyed (provides run status feedback). Optional (can be a NULL pointer)
[in]timeoutMaximum time interval (in ms) to wait for the thread to stop before force cancel. Only used when a thread mutex is provided for feedback. Must be equal or greater than 0
Precondition
assert( thread_id );
assert( quit_flag );
Returns
Upon successful return : ERR_OK (0)
Upon output error : An error code (<0)
Note
The thread should follow the follow structure
void thread_call( void )
{
pthread_mutex_lock( &thread_mutex );
while ( quit_flag == THREAD_RUN ){
//do stuff
}
pthread_mutex_unlock( &thread_mutex );
pthread_exit(NULL);
}
